# Ancestry Reference Panel Directory

This directory should contain ancestry reference panels for Module 7.

## Required Files

- `reference_panel.rds` - Main reference panel for ancestry estimation

## Reference Sources

Reference panels can be obtained from:
- 1000 Genomes Project (1KG)
- Human Genome Diversity Project (HGDP)
- gnomAD reference populations

## File Format

The reference panel should be an RDS file containing:
- Sample IDs with population labels
- Genotype data at ancestry-informative markers
