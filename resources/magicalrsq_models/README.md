# MagicalRsq-X Models Directory

This directory should contain trained models for MagicalRsq-X imputation quality filtering.

## Required Models

Models for different ancestries and imputation servers:
- TOPMed models
- All of Us/AnVIL models

## Source

MagicalRsq-X models can be obtained from:
https://github.com/sriramlab/MagicalRsq-X

## Usage

These models are used in Module 3 (Post-imputation QC) and Module 6 (Post-merge QC) to filter variants based on predicted imputation quality.
